# Скрипты Сопровождения

В этой папке находятся вспомогательные скрипты для разработки и релизного
процесса. Скрипты не должны содержать бизнес-логику перевода видео: основная
логика живет в пакете `src/translate_video`.

Текущий состав:

- `bump_version.py` — обновляет версию проекта в метаданных перед релизом.
