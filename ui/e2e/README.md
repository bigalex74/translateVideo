# Браузерные E2E-Тесты

В этой папке находятся Playwright-тесты, которые проверяют пользовательские
сценарии через реальный браузер. API-запросы в этих тестах мокируются на уровне
браузера, чтобы сценарии оставались быстрыми и не зависели от запущенного
backend-сервера.

Запуск:

```bash
npm run test:e2e
```
