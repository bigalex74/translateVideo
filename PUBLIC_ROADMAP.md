## 📋 Бэклог (одобрено, не в активном спринте)

| Фича | Статус | Примечание |
|------|--------|------------|
| Per-user JWT авторизация | 📋 Планируется | Большая задача — отдельный спринт |
| Загрузка видео по URL (YouTube, GDrive) | 📋 Планируется | Требует yt-dlp интеграции |
| Batch >10 проектов + очередь | 📋 Планируется | Требует Redis/Celery |
| CLI-клиент | 📋 Планируется | pip install translate-video-cli |
| PWA / мобильное приложение | 📋 Планируется | Долгосрочно |
| Email-уведомления о готовности | 📋 Планируется | SMTP интеграция |
| Глоссарий пользовательских терминов | 📋 Планируется | Для корпоративных |
| Экспорт в Word/PDF | 📋 Планируется | Для редакторов |

## 🎯 Долгосрочные цели
