# HTTP-Маршруты

В этой папке находятся FastAPI-маршруты пользовательских сценариев:

- `projects.py` — создание проекта, загрузка видео, список проектов, статус,
  сохранение сегментов и скачивание артефактов.
- `pipeline.py` — запуск пайплайна и отправка webhook-уведомлений о результате.
- `preflight.py` — предварительная проверка входного файла и окружения.

Маршруты должны оставаться тонкими: проверять вход, вызывать сервисы ядра и
возвращать стабильный JSON-контракт для CLI, UI и внешних интеграций.
