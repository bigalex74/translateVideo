# API-Слой

Папка содержит FastAPI-приложение, зависимости маршрутов и инфраструктуру
webhook-уведомлений. API-слой отвечает только за HTTP-контракт, валидацию
входных данных и передачу работы в ядро проекта.

Бизнес-логику пайплайна, хранения и провайдеров нужно размещать в профильных
модулях `core`, `pipeline`, `media`, `speech`, `translation`, `tts` и `render`.
