# Подгонка Таймингов

Пакет содержит логику подготовки перевода к естественной озвучке без ускорения
и обрезки аудио.

Главный принцип: сначала адаптировать текст под длительность сегмента, а уже
потом синтезировать речь. Если текст нельзя безопасно сократить, пайплайн
сохраняет полный смысл и ставит QA-флаг для проверки.

`natural.py` содержит детерминированный rewriter без локальных LLM. Он нужен как
безопасный базовый слой и точка подключения будущих облачных моделей.
