# Рендер

Сервисы рендера объединяют видео, исходное аудио, переведенную озвучку и
субтитры в итоговые артефакты. Первая реальная реализация может обернуть
устаревшее поведение MoviePy/ffmpeg, но пайплайн должен оставаться независимым
от провайдера.

`legacy.py` содержит MoviePy-рендерер закадровой озвучки.

## Политика Тайминга

Начиная с `TVIDEO-042` рендер не должен ухудшать восприятие перевода ради
жёсткого попадания в слот. По умолчанию он не ускоряет и не обрезает TTS-аудио.
Если фраза длиннее сегмента, аудио сохраняется полностью, сегмент получает
QA-флаг `render_audio_overflow`, а следующие реплики могут быть мягко сдвинуты
через `timeline_shifted`.

Ускорение доступно только при явном `allow_render_audio_speedup=True`.
Обрезка через `subclip` разрешена только при `allow_render_audio_trim=True`.
Оба режима считаются специальными настройками, не дефолтом продукта.
