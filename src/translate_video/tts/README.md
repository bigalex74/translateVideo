# TTS

TTS-сервисы синтезируют аудиофайлы для переведенных сегментов. Стратегия
голосов и сопоставление спикеров относятся к ядру; адаптер провайдера только
генерирует аудио.

`legacy.py` содержит адаптер `edge-tts` с базовой картой голосов по языкам.

## Политика Скорости

По умолчанию TTS работает на естественной скорости `+0%`. Ускорение голоса
ухудшает восприятие перевода, поэтому adaptive rate доступен только как явный
fast-режим через `allow_tts_rate_adaptation=True`.

Если фраза не помещается, пайплайн сначала запускает этап `timing_fit`, который
готовит более короткий текст до синтеза. Если безопасно сократить текст нельзя,
TTS сохраняет полный смысл и ставит QA-флаг `tts_overflow_natural_rate`.
