## Топ-3 системных паттерна

1. **«Пропасть между новичком и системой»** — приложение написано для тех кто уже знает что такое TTS, провайдеры, API-ключи. Новые пользователи теряются на старте. *Затронуто 3/5 пользователей.*
2. **«Невидимый процесс»** — пользователи не понимают что происходит во время обработки. Нет прогресса в %, нет ETA в UI (есть в overlay, но не везде). *Затронуто 4/5 пользователей.*
3. **«Профессиональный тупик»** — фрилансеры и DevOps упираются в лимиты (batch 10, нет очереди, нет per-user auth). *Затронуто 2/5 пользователей.*
